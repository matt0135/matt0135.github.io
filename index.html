OBJECT IN THE DARK

DGA1020

Object in the dark is a school project which required the representation of a lit object in the dark.



3 Main Materials Needed For This Project

  • Light Source/s
  • Camera
  • Darkroom

I, tackled this project at home using the light sources I could find around the house, the IPhone camera and I built a structure that mimics the purpose of a darkroom.

By doing this project I lerned various photography tricks including, the basics of ISO, Aperture and Shutter Speed.
